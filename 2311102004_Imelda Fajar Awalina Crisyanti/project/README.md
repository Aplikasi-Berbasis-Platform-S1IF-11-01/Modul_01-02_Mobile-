# Flutter Hello World

Project ini dibuat untuk tugas praktikum: menjalankan Flutter sampai menampilkan output **Hello World**.

## Cara Menjalankan

1. Pastikan Flutter SDK, Android Studio/Android SDK, dan VS Code sudah terinstal.
2. Extract file ZIP ini.
3. Buka terminal di folder `flutter_hello_world`.
4. Jalankan:

```bash
flutter pub get
flutter run
```

Jika ingin menjalankan di browser Chrome:

```bash
flutter run -d chrome
```

## Output

Aplikasi akan menampilkan teks **Hello World** di tengah layar dengan judul halaman **Flutter Hello World Page**.
